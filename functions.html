関数とは？

人間とはめんどくさがりな生き物です。
同じような文章が多くなってきたら、大昔の COBOL という言語のように
コピペする — そんなやり方はもうやりたくないんですね。

そこで登場するのが 関数 です！

関数というのは、「何か入れてくれれば、何か出すよ」っていう機械のようなものです。
自動販売機にお金を入れたらジュースが出てくる。
製氷機に水を入れたら氷が出てくる。
……お金を入れても反応しない、ぼったくり。それもまた関数です。

関数を書いてみよう

function myFunc(x)
    local y = x * 2
    return y
end

local answer = myFunc(2)
print(answer)

このコードを実行すると、4 という数字が出てくるはずです。

📖 どう動いているの？

  • myFunc という関数をコンピューターが作ってくれる（＝使えるようになる）
  • myFunc(2) で関数に 2 を渡す
  • x は 2 に置き換えられる
  • y は 2 * 2 = 4 になる
  • return でその 4 を返す
  • answer に返ってきた 4 が入る
  • print(answer) → 4 が表示される！

つまり関数は、「入れたら、返ってくる」もの。
そして、同じことを何度も書かずに済む便利な仕組みなのです。

main関数を使ってみよう

function main()
    print("えっと、0返してくれるよね…？")
end

main()

関数は 引数（＝中に渡すデータ） を渡さなくても動きます。
上のように () を空にすれば、ただ呼び出すだけでOK。
