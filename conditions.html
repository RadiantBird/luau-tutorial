if文（条件分岐）とは？

あなたは、外にいる時に雨が降っていたらどうしますか？
傘をさす、近くの建物に入る、そのまま浴びる…いろんな選択をしたと思います。

コンピューターも選択ができます。それが if文 です！

さっそくやってみましょう：

local x = 1

if x == 1 then
    print("xは1です")
end

新しく出てきたのが if 文です。
x == 1 の部分を 条件 といいます。

== は「左と右が同じかどうか」をチェックします。
（※ 厳密にはもっと奥深いけど、それはまた今度！）

コンピューターは嘘をつきません。
このまま実行してみると、xは1です と表示されるはずです。

では、x を 2 に変えてみましょう：

local x = 2

if x == 1 then
    print("xは1です")
end

おや？ 今回は何も言いません。

そう、条件が一致したときしか中身は実行されないんです！

でも、これだとちょっと不便ですね。
「そうじゃないとき」も知りたい。そんなときは else を使います。 

local x = 2

if x == 1 then
    print("xは1です")
else
    print("xは1じゃないよ")
end

実行すると、「xは1じゃないよ」と表示されると思います。

これは、条件に一致しなかったので else の方を実行したということです。

さらに、複数の条件を確かめたいときもありますよね？
たとえば「雨は降ってないけど曇ってるときはどうするか？」みたいな。

そんなときは else if を使います。

local x = 2

if x == 1 then
    print("xは1です")
else if x == 2 then
    print("xは2です")
else
    print("xは1でも2でもないよ")
end

実行すると xは2です と表示されると思います。

こんなふうに、コンピューターにも「選択」をさせることができるのが if文です。

条件が多くなりすぎて困るようになるのは、また後の話にしましょう _(:3」z)_
