テーブル（基礎）

テーブルって聞いた瞬間、何を思い浮かべましたか？多分、食べ物を食べるときに使うあの平面のことを思い浮かべたと思います。Luauのテーブルも、そんな感覚でデータを並べられちゃいます！

ここまでやってきたあなたなら、多分こんなことになったことない？

local x1 = 1
local x2 = 2
local x3 = 3
print(x1)
print(x2)
print(x3)

ああ、めんどくさい。変数も繰り返しができればいいのに。

そこでテーブルです！テーブルは、変数を並べたものだと今は思ってください。

使ってみよう:

local A = {1, 2, 3}
for i = 1, #A do
    print(A[i])
end

めっちゃコンパクトになりましたね！

  • 1行目で 1, 2, 3 を A に順番に入れます
  • 1番目から A の長さ分（3）繰り返すというのが i = 1, #A の意味です！
  • テーブルの前に # をつけると、特殊な場合を除いて長さを取得できます
  • 3行目は A の i 番目を喋って、ということになります

テーブルの外（この場合は 4 番目以降）を喋らせようとすると「そんなこと知らない！」（エラー）と赤文字で優しく教えてくれます。赤文字が出ても怖がらずに、翻訳してみると結構優しいんだなと気づけると思います。

これが for 文の使い方、スターになれたね！(・∀・)
